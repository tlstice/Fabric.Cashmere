Additional classes

TabComponent

Properties
Name Description

exactRouteMatch: boolean

If set to true, [routerLinkActiveOptions]="{exact: true}" is added

queryParams: Params

Add queryParams to the routerLink

routerLink: string[] | string

Router path that the tab routes to. If one tab uses the routerLink in a tab set, all must use the router link. Can be specified as '/path/2' or ['path', '2']

tabClick: EventEmitter<Event>

Emits when this tab is selected; use instead of (click) for click binding

tabContent: TemplateRef<unknown>

The template to be used when this tab is selected. Defaults to the content of this tab component. Not used when the tab set uses routing.

tabTitle: ''

Plain text title of the tab; for HTML support include a hc-tab-title element
Methods
tabClickHandler(event: Event)

TabSetComponent

Properties
Name Description

addContentContainer: boolean

Determines whether the tab set will create a router-outlet or ng-container for the tab content. If set to false, the app will need to add its own container. Defaults to true.

defaultTab: string | number

Zero-based numerical value specifying which tab to select by default, setting to none means no tab will be immediately selected. Defaults to 0 (the first tab).

direction: string

Specify direction of tabs as either horizontal or vertical. Defaults to vertical

selectedTabChange: EventEmitter<TabChangeEvent>

Emits when the selected tab is changed

tabContent: TemplateRef<unknown>

The content to be displayed for the currently selected tab. This is read from the tab when it is selected. Not used when this component uses routing.

tight: boolean

If true, condense the default padding on all included tabs. Defaults to false.
Methods
selectTab(tab: number | TabComponent)

Sets the currently selected tab by either its numerical index or TabComponent object
